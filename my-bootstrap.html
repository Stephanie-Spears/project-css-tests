Grid System in bootstrap

Lining up content evenly can be helped using bootstrap's grid system. it creates invisible grids to line things up. It contains two parts: rows and columns, which are both just div tags

  • ROW must be inside a container.
  • additionally, all columns MUST reside in a row. and each row may contain up to 12 columns.
  • Columns can include just about any HTML, not just text. So we can put panels, wells, headers, etc. inside them.
  • col-mediaquerysize-width
  • their class always begins with "col"
  • Columns are more customizable than rows. Their class has two additional pieces of information which denote what media queries can be applied to it, and how many units its width should be.
  • the format is: col media-query width
  • the media query portion can be either xs, sm, md, or lg sizes
  • the width part can be any number from 1-12. REMEMBER: a row can be ONLY 12 units wide, so a row can fit 12 columns of width 1, or 2 columns of width 6, or 3 columns of width 4.

Note: shortcut to move line (unselected) up or down--ctrl+arrow.
Ctrl + shift + d--duplicate current line

Column class names use the following media queries:

  • Phones: xs
  • Tablets: sm
  • Desktops: md
  • Large Desktops: lg

these prefixes denote how narrow a viewport may become before the colums stack upon one another--ie, how wide their breakpoint is.

The two most used categories of Bootstrap documentation that we'll reference are Components and CSS, both of which we can find in the navigation bar on the bootstrap website.

The CSS section includes entries for all the resources we can used to change the apperance or implement bootstrap's version of standard html elements like headings, tables, forms, grids, buttons, images, etc.

The COMPONENETS section has the elements bootstrap created on their own, like navigation bars, labels, jumbotrons, quick ways to alter the apperance of images, etc.

Any of these Bootstrap elements that have visual effects or animations require javascript (dropdown menus, modal windows, etc.)
